# プリセットとローカライズ

KIBAMaterialGUI は、表示ラベルやプリセットをプロジェクトに合わせて拡張できます。

## ラベルのローカライズ

標準では ShaderLab の表示ラベルを使います。プロジェクト側でローカライズテーブルを用意すると、現在の言語に合わせたラベルを表示できます。

推奨キーはプロパティ名です。

```text
_BaseColor = Base Color
_ShadowColor = Shadow Color
```

## プリセット

プリセットは複数のプロパティ値をまとめて適用したいときに使います。表面タイプ、ライティングモード、スタイル差分などに向いています。

## 診断

ローカライズキーやプリセット項目の不足は、診断 provider で作者向け警告として表示できます。
