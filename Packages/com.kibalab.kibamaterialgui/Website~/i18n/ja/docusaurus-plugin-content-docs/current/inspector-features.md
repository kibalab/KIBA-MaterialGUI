# インスペクター機能

KIBAMaterialGUI インスペクターは、大きなシェーダーでも目的の値をすばやく見つけられるように設計されています。

## 検索

検索対象はプロパティ名、表示ラベル、グループパス、ShaderLab Attribute 名です。

`[ShowIf]` で非表示になっているプロパティは、検索に一致しても表示されません。

## フィルター

フィルターはフラグ形式です。変更値、警告、テクスチャ、数値、カラーなどを組み合わせられます。

## グループ

グループヘッダーにはプロパティ数、mixed 状態、警告状態が簡潔に表示されます。グループメニューから折りたたみ、リセット、JSON コピー/貼り付けなどを実行できます。

## アニメーション

Unity のアニメーション記録またはプレビュー中のプロパティは、通常の MaterialEditor に近い色のヒントを表示します。記録モードでは右クリックメニューにキー追加/削除の項目も表示されます。
