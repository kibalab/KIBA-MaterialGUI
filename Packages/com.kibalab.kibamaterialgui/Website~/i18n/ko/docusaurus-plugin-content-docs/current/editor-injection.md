# 에디터 인젝션

에디터 인젝션은 KIBAMaterialGUI 인스펙터의 정해진 위치에 추가 UI를 삽입하는 기능입니다.

커스텀 렌더러가 "프로퍼티 한 줄"을 바꾸는 기능이라면, 에디터 인젝션은 "인스펙터 주변 UI"를 추가하는 기능입니다.

## 사용 예

- 툴바 아래에 프로젝트 전용 버튼 추가
- 특정 그룹 상단에 빠른 프리셋 버튼 배치
- 프로퍼티 아래에 도움말 표시
- 그룹 컨텍스트 메뉴에 커스텀 액션 추가
- 쉐이더 작성자용 진단 표시

## 확장 지점 고르기

- 단순히 한 위치에 IMGUI를 그리고 싶으면 hook injection
- 툴바 항목을 추가하려면 toolbar contributor
- 그룹 메뉴나 그룹 헤더 버튼이 필요하면 group contributor
- 경고/오류를 추가하려면 diagnostic provider
- 필터 칩을 추가하려면 filter provider
