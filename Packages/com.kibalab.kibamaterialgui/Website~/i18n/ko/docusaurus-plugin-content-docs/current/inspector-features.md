# 인스펙터 기능

KIBAMaterialGUI 인스펙터는 큰 쉐이더에서 원하는 값을 빠르게 찾고, 그룹 단위로 상태를 확인하도록 설계되어 있습니다.

## 검색

검색창은 다음 항목을 대상으로 합니다.

- 프로퍼티 이름
- 표시 라벨
- 그룹 경로
- ShaderLab Attribute 이름

`[ShowIf]` 조건으로 숨겨진 프로퍼티는 검색어가 일치해도 표시되지 않습니다.

## 필터

필터 드롭다운은 플래그 방식입니다. 여러 조건을 동시에 켤 수 있습니다.

- 변경된 값
- 진단 경고가 있는 값
- 텍스처
- 숫자
- 색상

## 그룹

그룹 헤더는 프로퍼티 수, mixed 상태, 경고 상태를 간결하게 표시합니다. 그룹 메뉴에서는 해당 그룹의 접기/펼치기, 리셋, JSON 복사/붙여넣기 같은 작업을 수행할 수 있습니다.

## 애니메이션 표시

Unity 애니메이션 녹화 또는 프리뷰 중인 프로퍼티는 Unity의 기본 머티리얼 필드처럼 애니메이션 색상 힌트를 표시합니다. 우클릭 메뉴에서는 녹화 모드일 때 키 추가/제거 관련 메뉴도 함께 나타납니다.
