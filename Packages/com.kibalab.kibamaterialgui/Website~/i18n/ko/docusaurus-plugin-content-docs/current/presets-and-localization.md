# 프리셋과 현지화

KIBAMaterialGUI은 머티리얼 편집자용 표시 라벨과 프리셋 데이터를 프로젝트에 맞게 확장할 수 있도록 설계되어 있습니다.

## 라벨 현지화

프로퍼티 라벨은 원본 ShaderLab 라벨을 기본값으로 사용합니다. 프로젝트에서 현지화 테이블을 제공하면 현재 언어에 맞는 라벨로 표시할 수 있습니다.

권장 키는 프로퍼티 이름입니다.

```text
_BaseColor = Base Color
_ShadowColor = Shadow Color
```

## 프리셋

프리셋은 여러 프로퍼티 값을 한 번에 적용해야 할 때 사용합니다. 예를 들어 표면 타입, 조명 모드, 스타일 변형을 머티리얼 작성자가 빠르게 바꾸도록 만들 수 있습니다.

프리셋을 적용하는 UI는 기본 인스펙터 기능 또는 에디터 인젝션으로 구성할 수 있습니다.

## 진단

현지화 키나 프리셋 항목이 누락된 경우, 진단 제공자를 통해 작성자용 경고로 표시할 수 있습니다.
